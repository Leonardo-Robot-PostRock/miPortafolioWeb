# Habilidades de Leonardo Puebla

Soy un desarrollador frontend con perfil full stack, con más de 3 años de experiencia transformando sistemas complejos y aplicaciones legacy en soluciones modernas, escalables y bien documentadas. Mi enfoque combina calidad de código, claridad arquitectónica y resolución rápida de incidencias.

## Habilidades Técnicas

### Frontend
- **React**, **Next.js**, **Redux**: Desarrollo de aplicaciones web modernas con componentes reutilizables y estado centralizado.
- **Svelte** / **SvelteKit**: Creación de interfaces dinámicas y soporte offline con PouchDB.
- **Angular**: Construcción de aplicaciones interactivas (ej. app del Mundial de Qatar 2022).
- **React Native**: Actualización y desarrollo de aplicaciones móviles.
- **AG Grid**: Implementación de tablas complejas para gestión de datos.

### Backend
- **Node.js** / **Express**: Desarrollo de APIs RESTful y endpoints de sincronización.
